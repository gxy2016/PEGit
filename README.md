# PEGit

> Prompt Engineering & Git - Enterprise Prompt Management System

## 📅 项目启动日期
2026-04-26

## 📖 项目描述
本项目主要参考 Git 代码管理功能，针对企业中的 Prompt 提示词进行全生命周期管理。
PEGit 旨在解决企业级 AI 应用中提示词版本混乱、效果难以追踪、协作效率低下等问题，提供一套标准化、自动化的提示词管理解决方案。

## ✨ 核心功能
- **提示词注册**：支持提示词的创建、分类、标签管理与元数据描述
- **内容存储**：安全存储提示词模板、变量定义及使用上下文
- **版本管理**：类似 Git 的版本控制，支持分支、合并、回滚与 Diff 对比
- **效果评估**：内置测试框架，支持多模型输出对比、质量评分与效果追踪
- **端侧 CLI 工具**：提供 `pegit` 命令行工具，可在任何项目中实现 `pegit add`、`pegit commit`、`pegit push` 等提示词管理基本能力
